We cannot save the password in plain text ("francoisthebest") otherwise if we get hacked the hackers would know what the password is. So we counter this by encrypting the password before saving it to our database we will be using SHA-256 for this example: This password will be Encrypted to SHA-256 then compare if the SHA-256 values are the same as the database. "francoisthebest" is now saved as "7b86e763af0073ba3344874559885a3cb81b509ca88f131d52325b9da4a05d26" on our database
